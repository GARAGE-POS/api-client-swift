# Swift6 API client for KarageAPIClient

Welcome to the Karage API! This API provides programmatic access to manage your business operations including accounts, services, orders, subscriptions, notifications, and more.

## Authentication

The Karage API uses **OAuth 2.0** with the Authorization Code flow (with PKCE) for authentication.

### Getting Started

1. **Obtain Client Credentials**: Register your application to receive a `client_id`
2. **Authorization Request**: Redirect users to the authorization endpoint to grant access
3. **Token Exchange**: Exchange the authorization code for an access token
4. **API Requests**: Include the access token in the `Authorization` header as `Bearer <token>`

### OAuth2 Endpoints

| Endpoint | URL |
|----------|-----|
| Authorization | `/oauth/authorize` |
| Token | `/oauth/token` |

### Available Scopes

| Scope | Description |
|-------|-------------|
| `email` | Access to your email address |
| `profile` | Access to your profile information |
| `offline_access` | Allows refresh tokens for long-lived access |

### Example Authorization Request

```
GET /oauth/authorize?
  response_type=code&
  client_id=YOUR_CLIENT_ID&
  redirect_uri=YOUR_REDIRECT_URI&
  scope=email profile offline_access&
  code_challenge=YOUR_CODE_CHALLENGE&
  code_challenge_method=S256
```

> **Tip**: Use the **Authorize** button above to test the API interactively with your credentials.

## Installation

### Swift Package Manager (Recommended)

Add this package to your Xcode project:

1. In Xcode, go to **File** → **Add Package Dependencies...**
2. Enter the repository URL: `https://github.com/GARAGE-POS/api-client-swift.git`
3. Choose the version or branch you want to use
4. Click **Add Package**

Or add it to your `Package.swift` dependencies:

```swift
dependencies: [
    .package(url: "https://github.com/GARAGE-POS/api-client-swift.git", from: "1.0.0")
]
```

Then add `KarageAPIClient` to your target dependencies:

```swift
.target(
    name: "YourTarget",
    dependencies: [
        "KarageAPIClient",
        .product(name: "OpenAPIURLSession", package: "swift-openapi-urlsession")
    ]
)
```

## Quick Start

```swift
import KarageAPIClient
import OpenAPIURLSession

// Create a client with your API base URL
let client = Client(
    serverURL: try Servers.server1(),  // https://api-qa1.karage.co
    transport: URLSessionTransport()
)

// Example: Create system compliance
let input = Operations.CreateSystemCompliance.Input(
    headers: .init(
        X_hyphen_Karage_hyphen_Request_hyphen_ID: "unique-request-id",
        X_hyphen_Account_hyphen_ID: 1,
        X_hyphen_Location_hyphen_ID: 1
    ),
    body: .json(.init(/* your request data */))
)

do {
    let response = try await client.CreateSystemCompliance(input)
    
    switch response {
    case .created(let success):
        // Handle successful response
        if case .json(let compliance) = success.body {
            print("Created: \(compliance)")
        }
    case .badRequest:
        print("Bad request")
    case .unauthorized:
        print("Unauthorized")
    default:
        print("Other response")
    }
} catch {
    print("Error: \(error)")
}
```

### Sync with OAuth Configuration

If you manage OAuth configuration separately in your app, sync it when environment changes:

```swift
import KarageAPIClient

class AppEnvironmentManager {
    
    func switchEnvironment(to environment: Environment) {
        // 1. Switch API environment
        EnvironmentManager.shared.switchToEnvironment(environment)
        
        // 2. Update your OAuth config to match
        updateOAuth2Config(environment: environment)
        
        // 3. Logout user to force re-authentication
        logoutUser()
    }
    
    private func updateOAuth2Config(environment: Environment) {
        // Update your OAuth2Config.json issuer to match:
        let issuer = environment.baseURL
        
        guard let configURL = getOAuthConfigURL() else { return }
        guard let data = try? Data(contentsOf: configURL) else { return }
        guard var json = try? JSONSerialization.jsonObject(with: data) as? [String: Any] else { return }
        
        json["issuer"] = issuer
        
        if let updatedData = try? JSONSerialization.data(withJSONObject: json, options: .prettyPrinted) {
            try? updatedData.write(to: configURL)
            print("✅ OAuth config synced to \(environment.rawValue)")
        }
    }
    
    private func getOAuthConfigURL() -> URL? {
        // Your OAuth config location logic
        return nil
    }
}
```

## Authentication with Bearer Token

To add OAuth2 bearer token authentication:

```swift
import OpenAPIURLSession
import Foundation

// Create a custom middleware that adds the Authorization header
struct AuthenticationMiddleware: ClientMiddleware {
    let token: String
    
    func intercept(
        _ request: HTTPRequest,
        baseURL: URL,
        operationID: String,
        next: (HTTPRequest, URL) async throws -> HTTPResponse
    ) async throws -> HTTPResponse {
        var request = request
        request.headerFields.append(.init(name: "Authorization", value: "Bearer \(token)"))
        return try await next(request, baseURL)
    }
}

// Use it with your client
let client = Client(
    serverURL: try Servers.server1(),
    transport: URLSessionTransport(),
    middlewares: [AuthenticationMiddleware(token: "YOUR_ACCESS_TOKEN")]
)
```
